# Tailwind 언어 취합 내용 (abc, ㄱㄴㄷ 순서)

## 개념

- 테일윈드 (Tailwind)는 유틸리티 기반 (Utility-first) 개발 방식을 제공
- 필요한 부품을 조립하는 방식으로 사용자 입맞에 맞게 커스터마이징이 가능
- 모바일 퍼스트 (Mobile-first) 디자인
- View 중심의 쉬운 디자인 변경
- 다만 HTML 요소에 Class 속성이 혼란스럽게 되어있음
- 항상 앞에 'class'를 선언하고 작성을 해야 함.
